Java სინტაქსი

წინა თავებში, ჩვენ შევქმენით Java ფაილი, დასახელებით Main.java და გამოვიყენეთ შესაბამისი კოდი, რათა კონსოლში დაგვეწერა "Hello World"

მაგალითის განმარტება

ჯავაში კოდის თითოეული ხაზი მოთავსებული უნდა იყოს კლასის შიგნით. ჩვენს მაგალითში, კლასს ვუწოდეთ Main. კლასის სახელის პირველი ასო ყოველთვის მაღალ რეგისტრში უნდა იყოს (უნდა დაიწეროს დიდი ასოთი)

ჯავა რეგისტრის მიმართ სენსიტიური. ჯავაში დასახელებებს "MyClass" და "myclass" აქვთ განსხვავებული მნიშვნელობები

ჯავას ფაილის სახელი ყოველთვის უნდა იყოს კლასის სახელის იდენტური. როდესაც ვინახავთ ფაილს, იგი უნდა შევინახოთ კლასის სახელით და დავუმატოთ გაფართოება .java. იმისათვის, რომ აღნიშნული კოდი გაეშვას თქვენს კომპიუტერში, საჭიროა ჯავა დაინსტალირებული იყოს უშეცდომოდ.

main მეთოდი

main() მეთოდის გამოძახება ჯავაში აუცილებელია, ეს არის ჯავაში ინტეგრირებული მეთოდი, რომელიც უზრუნველყოფს კოდის გაშვებას:

ნებისმიერი კოდი main() მეთოდის შიგნით აღსრულდება. main მეთოდის წინ და შემდეგ არსებული საკვანძო სიტყვები განმარტებული იქნება მომდევნო თავებში

დაიმახსოვრეთ, რომ ჯავაში კლასის დასახელება და ფაილის სახელი უნდა იყოს იდენტური და კოდი უნდა შეიცავდეს main() მეთოდს

System.out.println()

main() მეთოდის შიგნით, ჩვენ შეგვიძლია გამოვიყენოთ println() მეთოდი, რათა სქრინზე გამოვიტანოთ სასურველი ტექსტი.

ფიგურული ფრჩხილები {} აღნიშნავს კოდის ტანის დასაწყისსა და დასასრულს.

System Java-ში ჩაშენებული კლასია, რომელიც შეიცავს ისეთ წევრებს, როგორიცაა მაგ. out, რაც აღნიშნავს სიტყვა "output"-ის შემოკლებას. println() მეთოდი, კი არის შემოკლებული "print line", და გამოიყენება სქრინზე მნიშვნელობის დასაბეჭდად.

არ იდარდოთ System, out და println() საკვანძო სიტყვებზე. მთავარია გახსოვდეთ, რომ აღნიშნული კომბინაცია გამოიყენება კონსოლში სასურველი მნიშვნელობის დასაბეჭდად.

ამასთან, გახსოვდეთ, რომ კოდი უნდა დასრულდეს წერტილ-მძიმით (;).
