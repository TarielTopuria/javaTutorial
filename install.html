Java ინსტალაცია

ზოგიერთ კომპიუტერში შესაძლოა ჯავა უკვე დაინსტალირებული იყოს.

იმისათვის რომ შევამოწმოთ ჩვენს მოწყობილობაში ჩაწერილია თუ არა აჯვას რომელიმე ვერსია, უნდა გავხსნათ კონსოლი cmd.exe და შევიყვანოთ შესაბამისი ბრძანება

იმ შემთხვევაში თუ თქვენს მოწყობილობაზე დაინსტალირებულია ჯავას რომელიმე ვერსია, კონსოლში გამოვა შესაბამისი განმარტება

იმ შემთხვევაში თუ თქვენს მოწყობილობაზე არ არის დაინსტალირებული ჯავას რომელიმე ვერსია, აღნიშნულის გადმოწერა შესაძლებელია oracle.com-დან

ჩვენს შემთხვევაში კოდი დაწერილია text editor-ში, თუმცა კოდის წერა შესაძლებელია ისეთ ინტეგრირებულ გარემოებში, როგორიცაა მაგ. IntelliJ IDEA, Netbeans ან Eclipse, რომლებიც ძალიან კარგი გარემოებია ჯავას მრავალ ფაილთან სამუშაოდ.

Java-ს ინსტალაცია Windows-ზე

იმისთვის, რომ Java დავაყენოთ ვინდოუსის ოპერაციულ სისტემაში, საჭიროა:

  • შეხვიდეთ "System Properties" (Control Panel > System and Security > System > Advanced System Settings)
  • მონიშნოთ "Environment variables"
  • მონიშნოთ "Path" ცვლადი "System variables"-ში და დააჭიროთ "Edit" ღილაკს
  • დააჭიროთ "New" ღილაკს და დაამატოთ path იმის მიხედვით, სადაც Java-ს \bin ფოლდერი მდებარეობს. default-ად Java ჯავა ინსტალირდება C:\Program Files\Java\jdk-11.0.1 (თუ სხვა დანიშნულებას თავად არ მიუთითებთ). ასეთ შემთხვევაში, მოგიწევთ, რომ დაამატოთ path შემდეგი მისამართით C:\Program Files\Java\jdk-11.0.1\bin
  • საბოლოოდ, კონსოლში შეიყვანეთ ბრძანება java -version, რათა დარწმუნდეთ, რომ ჯავა წარმატებით დაინსტალირდა

Java - კოდის წერის საწყისები

კავასი კოდის წერა იწყება კლასის გამოცხადებით, რომლის სახელიც ფაილის იდენტური უნდა იყოს.

შევქმნათ პირველი ჯავა ფაილი, დასახელებით Main.java

აღნიშნული კოდის გაშვების შემდეგ კონსოლში უნდა დაიბეჭდოს შეტყობინება ტექსტით "Hello World":

თუ ზემოთ დასახელებული კოდის შინაარსი დეტალურად არ გესმით, არაუშავს. აღნიშნული კოდი განხილული და განმარტებული იქნება მომდევნო თავშებში დეტალურად. ახლა კი ორიენტირება გავაკეთოთ იმაზე თუ როგორ უნდა გაეშვას კოდი

Notepad-ში შეინახეთ აღნიშნული კოდის შემცველი ფაილი დასახელებით Main.java. აღნიშნულის შემდეგ გახსენით კონსოლი (cmd), მიმართეთ დირექტორიას, რომელშიც შენახული გაქვთ ფაილი და კონსოლში ჩაწერეთ ბრძანება "javac Main.java":

აღნიშნული ბრძანები გაშვება დააკომპილირებს თქვენს კოდს (გადაიყვანს მანქანისთვის გასაგებ ენაზე). იმ შემთხვევაში თუ კოდში შეცდომებია დაშვებული, კონსოლში გამოვა შესაბამისი შეტყობინება, ხოლო თუ არ არის რაიმე შეცდომა, ასეთ შემთხვევაში კონსოლი გადავა ახალ ხაზზე, სადაც უნდა შეიყვანოთ ბრძანება java Main, რათა გაეშვას კოდი

კონსოლში დაბრუნებული შეტყობინება უნდა იყოს შემდეგი:
