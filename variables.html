Java ცვლადები

ცვლადები არიან ერთგვარი კონტეინერები, რომლებიც ინახავენ მონაცემთა მნიშვნელობას

ჯავაში არსებობს მრავალი სახის ცვლადი, მაგალითად:

  • String - ინახავს ტექსტს, როგორიცაა მაგალითად "Hello". String ცვლადში შენახული მნიშვნელობა მოქცეულია ბრჭყალებში;
  • int ტიპის ცვლადი ინახავს მთელ რიცხვებს, როგორც დადებითი ასევე უარყოფითი მნიშვნელობით, თუმცა მას არ შეუძლია შეინახოს ათწილადები;
  • float ცვლადი კი ინახავს ათწილადებს, როგორც დადებითი ასევე უარყოფითი მნიშვნელობით;
  • char ტიპის ცვლადი ინახავს ერთ გარკვეულ სიმბოლოს, მაგ. 'a' ან 'b', რომლებიც განსხვავებით String ტიპის ცვლადისგან, მოქცეულია ერთმძიმიან ბრჭყალებში;
  • boolean ტიპის ცვლადი კი ინახავს მხოლოდ ორ შესაძლო მნიშვნელობას, ჭეშმარიტებას ან მცდარს (true ან false);

ცვლადების გამოცხადება (შექმნა)

ცვლადის გამოსაცხადებლად, საჭიროა განსაზღვროთ მონაცემთა ტიპი და მიანიჭოთ მას მნიშვნელობა:

როდესაც დასახელდება ჯავაში არსებული მონაცემთა ტიპი (მაგ. int ან String) და განისაზღვრება მისი სახელი (მაგ. x ან name), ამ დროს ტოლობის ნიშანი გამოიყენება გარკვეული მნიშვნელობის მინიჭებისთვის.

იმისათვის, რომ შეიქმნას ცვლადი, რომელიც შეინახავს გარკვეულ ტექსტს, უნდა გამოვიყენოთ კოდი, შემდეგი მაგალითის ანალოგიურად:

იმისათვის, რომ შეიქმნას ცვლადი, რომელიც შეინახავს მთელ რიცხვს, უნდა გამოვიყენოთ კოდი, შემდეგი მაგალითის ანალოგიურად:

ჯავაში შესაძლებელია ცვლადის გამოცხადება, მისთვის მნიშვნელობის მინიჭების გარეშე, იმ მიზნით, რომ მნიშვნელობა მიენიჭება მოგვიანებით. მაგალითად:

იმ შემთხვევაში თუ უკვე შექმნილ ცვლადს მიანიჭებთ ახალ მნიშვნელობას, ახალი მნიშვნელობა გადაეწერება ძველს. მაგალითად:

სხვა ტიპები

წინამდებარე მაგალითში მოცემულია მაგალითები თუ როგორ უნდა გამოცხადდეს სხვადასხვა ტიპის ცვლადები

ცვლადების გამოსახვა

println() მეთოდი ხშირად გამოიყერნება იმისათვის, რომ ეკრანზე გამოვსახოთ ცვლადის მნიშვნელობა

იმისათვის, რომ დავაკავშიროთ ტექსტი და ნებისმიერი ცვლადი გამოიყენება "+" სიმბოლო

თქვენ ასევე შეგიძლიათ გამოიყენოთ "+" სიმბოლო, რათა დააკავშიროთ ერთი ცვლადი მეორესთან

რიცხვითი მნიშვნელობის ცვლადებთან მიმართებით "+" სიმბოლო ფუნქიონირებს, როგორც მათემატიკური ოპერატორი (ყურადღება მიაქციეთ, რომ ქვემოთ მოცემულ მაგალითში გამოიყენება int ტიპის ცვლადი):

ზემოთ მოცემულ მაგალითში შეამჩნევდით, რომ:

  • x ცვლადი ინახავს მნიშვნელობას, რომელიც ტოლია 5-ის
  • y ცვლადი ინახავს მნიშვნელობას, რომელიც ტოლია 6-ის
  • გამოყენებულ მეთოდს println()-ს კი გამოაქვს მნიშვნელობა, რომელიც ტოლია x+y-ის და არის 11

მრავალი ცვლადის გამოცხადება

ერთი და იმავე ტიპის რამდენიმე ცვლადის გამოსცხადებლად შესაძლებელია გამოვიყენოთ მძიმით გამოყოფილი კოდი. მაგ. ნაცვლად დაწერისა, რომ

  • int x = 5
  • int y = 6
  • int z = 50

შეგვიძლია დავწეროთ, რომ int x = 5, y = 6, z = 50;

ასევე შესაძლებელია, რომ მრავალ ცვლადს მივანიჭოთ ერთიანი მნიშვლეობა ერთ ხაზში

იდენტიფიკატორები

ყველა ცვლადი უნდა იდენტიფიცირდებოდეს საკუთარი უნიკალური სახელით

აღნიშნულ უნიკალურ სახელებს ეწოდება იდენტიფიკატორები

იდენტიფიკატორები შესძლებელია იყოს მოკლე დასახელებები (მაგ. x და y) ან მეტად აღწერილი (მაგ. age, sum, totalVolume).

რეკომენდირებულია, რომ გამოვიყენოთ აღწერილობითი სახელები, რათა კოდი გახდეს მეტად გასაგები და ადვილად გამოსაყენებელი:

არსებობს ზოგადი წესები, რომლებიც გამოიყენება ცვლადებისთვის სახელის დარქმევისას:

  • ცვლადის სახელები შესაძლებელია შეიცავდეს ასოებს, რიცხვებს, ქვედა ტირესა და დოლარის ნიშანს;
  • ცვლადის სახელები უნდა დაიწყოს ასოთი;
  • ცვლადის სახელების წერა უნდა დაიწყოს დაბალი რეგისტრის ასოთი და არ შეიძლება შეიცავდეს გამოტოვებას (whitespace);
  • ცვლადის სახელები, ასევე, შესაძლებელია დაიწყოს $ და _;
  • ცვლადის სახელები არის რეგისტრის მიმართ სენსიტიური ("myVar" და "myvar" სხვადასხვა ცვლადებია);
  • ჯავაში არსებობს დარეზერვებული სიტყვები, რომლებიც არ უნდა დაერქვას ცვლადს (მაგ. int ან boolean);
