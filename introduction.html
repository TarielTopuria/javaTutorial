Java შესავალი

რა არის ჯავა?

ჯავა მსოფლიოში პოპულარული პროგრამირების ენაა, რომელიც 1995 წელს შეიქმნა.
ჯავას შექმნილია კომპანია Oracle-ის მიერ და იგი გამოიყენება 3 მილიარდზე მეტ მოწყობილობაში.
იგი გამოიყენება:
  • მობილურ აპლიკაციებში
  • დესკტოპ აპლიკაციებში
  • ვებ აპლიკაციებში
  • ვებ სერვერებთან სამუშაოდ
  • კომპიუტერულ თამაშებში
  • მონაცემთა ბაზებთან მუშაობისთვის
  • და ა.შ.

რატომ Java?

  • ჯავას შეუძლია იმუშაოს მრავალ პლატფორმაზე (Windows, Mac, Linux, Paspberyy Pi და ა.შ.)
  • იგი არის მსოფლიოში ერთ-ერთი ყველაზე პოპულარული ენაა
  • მისი შესწავლა და გამოყენება მარტივია
  • უფასოა
  • უსაფრთხოა, სწრაფი და ძლიერი
  • მასზე მუშაობს მილიონობით დეველოპერი
  • ის არის ობიექტზე ორიენტირებული პროგრამირების ენა, რომელიც უზრუნველყოფს პროგრამის მარტივ სტრუქტურას და კოდის მრავალჯერ გამოყენებადობას, რაც ამცირებს დეველოპმენტის ხარჯებს
  • ჯავა დგას ახლოს C++ და C#, რაც დეველოპერებისთვის ამარტივებს აღნიშნული ენების შესწავლას.
